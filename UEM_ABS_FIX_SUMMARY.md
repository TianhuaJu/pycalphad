### 2. 发现系统性问题

通过调试工具发现：在 Al-Cr-Ni LIQUID 相中，**α_CR^NI 在整个成分范围内都是负数**！

**测试数据（T=1700K, X(AL)=X(CR), X(NI)=0.1~0.9）：**

| X(NI) | d_NIAL | d_NICR | α_CR^NI | 状态 |
|-------|--------|--------|---------|------|
| 0.10 | 0.016626 | -0.864829 | -0.046545 | ⚠️ 负数 |
| 0.20 | 0.016626 | -0.864829 | -0.046545 | ⚠️ 负数 |
| ... | ... | ... | ... | ... |
| 0.90 | 0.016626 | -0.864829 | -0.046545 | ⚠️ 负数 |

**100% 的测试点都出现负贡献系数！**

### 3. 根本原因分析

**计算逻辑：**
```
d_NIAL = (2/RT) * sum(L_odd_NIAL) = +0.016626
d_NICR = (2/RT) * sum(L_odd_NICR) = -0.864829  ← 负数！
d_sum = d_NIAL + d_NICR = -0.848203

α_CR^NI = (d_NIAL / d_sum) * exp(-d_NICR)
        = (0.016626 / -0.848203) * exp(0.864829)
        = -0.0196 * 2.375
        = -0.046545  ← 负数！
```

**问题所在：**
- d_NICR 为负是因为 L1(NI,CR) 参数为负值
- 但 UEM 公式要求 d_ki >= 0（代表无限稀释性质的差异）
- **缺少绝对值运算！**

## ✅ 用户指出关键

用户指出：**"性质差的计算要取绝对值"**

```
d_ki_num = g_i_inf_in_k - g_k_inf_in_i
d_kj_num = g_j_inf_in_k - g_k_inf_in_j
```

**必须取绝对值！**

## 🔧 修复方案

### UEM 理论基础

根据 UEM 原始理论：

```
d_ki = (1/RT) * |g_i^∞(in k) - g_k^∞(in i)|
```

对于 Redlich-Kister 模型：
```
g_i^∞(in k) = L0 + L1 + L2 + L3 + ...
g_k^∞(in i) = L0 - L1 + L2 - L3 + ...

差值 = 2 * (L1 + L3 + L5 + ...)

d_ki = (2/RT) * |sum(L_odd)|  ← 关键：取绝对值！
```

### 代码修改

#### 1. pycalphad/uem1_Model.py (第 148 行)

**修改前：**
```python
d_ki = (S(2) / (v.R * v.T)) * sign_correction * Add(*odd_L_terms)
```

**修改后：**
```python
from symengine import Abs
d_ki = Abs((S(2) / (v.R * v.T)) * sign_correction * Add(*odd_L_terms))
```

#### 2. evaluate_uem_numerical.py (第 91 行)

**修改前：**
```python
d_ki = (2.0 / (R * temperature)) * sign_correction * sum(odd_L_values)
```

**修改后：**
```python
# ✅ 关键修正：取绝对值！
# d_ki = (1/RT) * |g_i^∞(in k) - g_k^∞(in i)|
d_ki = abs((2.0 / (R * temperature)) * sign_correction * sum(odd_L_values))
```

## 🎓 经验总结

### 关键教训

1. **严格遵循理论公式**
   - UEM 理论明确要求取绝对值
   - 不能因为实现简便而忽略

2. **符号处理的重要性**
   - Redlich-Kister 参数有符号约定
   - 必须正确处理参数顺序
   - 最后取绝对值确保物理合理性

3. **调试工具的价值**
   - 打印中间变量能快速定位问题
   - 批量测试能发现系统性错误

4. **用户反馈的重要性**
   - 用户观察到的"化学势异常"是关键线索
   - 用户指出"性质差要取绝对值"直击要害

### UEM 公式正确实现

```python
# 第1步：计算无限稀释性质差（带符号）
d_ki_num = g_i_inf_in_k - g_k_inf_in_i

# 对于 R-K 模型：
# g_i_inf_in_k = L0 + L1 + L2 + L3 + ...
# g_k_inf_in_i = L0 - L1 + L2 - L3 + ...
# d_ki_num = 2 * (L1 + L3 + L5 + ...)

# 第2步：考虑参数顺序（符号修正）
d_ki_corrected = sign_correction * d_ki_num

# 第3步：归一化并取绝对值（关键！）
d_ki = abs(d_ki_corrected / (R * T))

# 第4步：计算贡献系数
α_i^k = (d_kj / (d_ki + d_kj)) * exp(-d_ki)  # 现在 d_ki >= 0, α >= 0
```
